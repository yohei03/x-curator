# X Timeline Curator

## 概要
Xの投稿を多層クエリで収集し、AIで評価するシステムの検証プロジェクト。

## MCP サーバー
xmcp が http://127.0.0.1:8000/mcp で起動している。
X API の操作はすべてこの MCP サーバー経由で行う。
コードを書かず、MCP ツールを直接呼び出して検証すること。

## スキル
@SKILL.md を参照すること。
設計の詳細・評価軸・プロンプト・ソース設定はすべてここに集約されている。

## 設定ファイル
@config/sources.json が現在のソース設定。
検索KW・List・トレンド設定はここを参照する。

## 出力
検証結果は out/ ディレクトリに保存する。
ファイル名は {YYYY-MM-DD}_{内容}.md の形式。
